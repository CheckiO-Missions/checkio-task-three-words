Що робити далі?
Якщо counter вже дорівнює 3, це значить є три слова підряд у рядку і немає потреби продовжувати підрахунок - просто поверни True. 

count = 0
for w in words.split():
    if w.isalpha():
        count += 1
        if count == 3:
            return True
    else:
        count = 0
То як має виглядати моє остаточне рішення?
Якщо не залишилося слів для перевірки (цикл завершився), це значить рядок не містить три слова підряд, тож повертай False. Оберни весь код у вираз функції. 

def checkio(words: str) -> bool:

    count = 0
    for w in words.split():
        if w.isalpha():
            count += 1
            if count == 3:
                return True
        else:
            count = 0
            
    return False

З використанням модуля регулярних виразів re і пошуком в рядку патерну з трьома сукупностями літер і пробілів між ними. 

import re


def checkio(words):

    return bool(re.search('\D+ \D+ \D+', words))
Створено три списки слів: починаючи з першого, другого і третього. Вони зібрані разом по індексу, кожні три слова з'єднані в одне без пробілів і здійснюється перевірка, чи результат цього з'єднання є словом. 

def checkio(words):

    d = words.split()

    return any(''.join(values).isalpha() for values in zip(d, d[1:], d[2:]))

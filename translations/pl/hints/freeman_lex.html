Nie mam pojęcia, jak zacząć rozwiązywać tę misję

💡 Ogólny pomysł polega na iterowaniu po podzielonym ciągu i liczeniu tylko słów z literami.

Od jakiego instrumentu powinienem zacząć?

Musi Pan pobrać oddzielne słowa z ciągu, a następnie iterować po nich. Proszę użyć metody string str.split() do pierwszego zadania i for statement do utworzenia pętli. 

for w in words.split():
    ...

Potrzebuję pomocy, aby kontynuować misję.

Aby rozróżnić słowa i liczby, proszę użyć metod łańcuchowych str.isalpha() i str.isdigit(). Dla tej misji wystarczy jedna metoda. Proszę użyć pierwszej z nich. Zwraca ona True jeśli ciąg zawiera tylko litery. Proszę również użyć if statement. 

if w.isalpha():
    ...
else:
    ...

Zniknąłem w połowie. Proszę o pomoc!

Proszę zadeklarować zmienną counter z wartością początkową 0 przed pętlą. Jeśli isalpha() dla bieżącego słowa to True, proszę zwiększyć counter, jeśli False - proszę zresetować licznik, ponieważ trzeba zliczać tylko słowa po kolei. 

count = 0
...
if w.isalpha():
    count += 1
    ...
else:
    count = 0

Utknąłem. Proszę o małą podpowiedź.

Ponieważ musi Pan sprawdzać każde słowo, dopóki counter nie osiągnie wartości 3 lub nie pozostanie żadne słowo, proszę wykonać powyższe sprawdzenie wewnątrz pętli for. 

count = 0
for w in words.split():
    if w.isalpha():
        count += 1
        ...
    else:
        count = 0
...

Co robić dalej?

Jeśli Państwa counter jest już równe 3, oznacza to, że w ciągu znajdują się trzy kolejne słowa i nie ma potrzeby kontynuowania liczenia - wystarczy zwrócić True. 

count = 0
for w in words.split():
    if w.isalpha():
        count += 1
        if count == 3:
            return True
    else:
        count = 0
...

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

jeśli nie pozostało żadne słowo do sprawdzenia (pętla się kończy), oznacza to, że ciąg nie zawiera trzech słów po kolei, więc proszę zwrócić False. Proszę zawinąć cały kod w wyrażenie funkcyjne. 

def checkio(words: str) -> bool:

    count = 0
    for w in words.split():
        if w.isalpha():
            count += 1
            if count == 3:
                return True
        else:
            count = 0
            
    return False

Jasne, proszę łapać!

Używając modułu wyrażeń regularnych i wzorca wyszukiwania z trzema sekwencjami liter i białymi znakami między nimi. 

import re


def checkio(words: str) -> bool:

    return bool(re.search('\D+ \D+ \D+', words))

Proszę utworzyć trzy listy słów: pierwszą, drugą i trzecią. Następnie proszę je połączyć, połączyć każde trzy słowa w jedno bez spacji i sprawdzić, czy jest to słowo. 

def checkio(words: str) -> bool:

    d = words.split()

    return any(''.join(values).isalpha() for values in zip(d, d[1:], d[2:]))

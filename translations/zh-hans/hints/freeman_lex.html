如果没有剩余的字需要检查（循环结束），这意味着该字符串没有连续的三个字，所以返回False 。将整个代码包裹在函数表达式中。 

def checkio(words: str) -> bool:

    count = 0
    for w in words.split():
        if w.isalpha():
            count += 1
            if count == 3:
                return True
        else:
            count = 0
            
    return False

使用正则表达式模块，用三个字母序列和它们之间的空白来搜索模式。 

import re


def checkio(words: str) -> bool:

    return bool(re.search('\D+ \D+ \D+', words))

创建三个词的列表：从第一个、第二个和第三个。然后将它们组合在一起，将每三个词连接成一个没有空白的词，并检查它是否是一个词。 

def checkio(words: str) -> bool:

    d = words.split()

    return any(''.join(values).isalpha() for values in zip(d, d[1:], d[2:]))

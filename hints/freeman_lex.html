if there are no word remains to check (loop ends), it means the string does not have three words in succession, so return False. Wrap the whole code in function expression. 

def checkio(words: str) -> bool:

    count = 0
    for w in words.split():
        if w.isalpha():
            count += 1
            if count == 3:
                return True
        else:
            count = 0
            
    return False

Creating three lists of words: from the first, the second and the third. Then zippind them together, join each three words in one without whitespace and check if it is a word. 

def checkio(words):

    d = words.split()

    return any(''.join(values).isalpha() for values in zip(d, d[1:], d[2:]))
